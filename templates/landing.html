Features

必要な機能が
すべて揃っています

シンプルなのに、ちゃんと使える。日常の Bluesky 運用をかしこくサポートします。

📅

日時指定で予約投稿

投稿したい日時をカレンダーから選ぶだけ。バックグラウンドで自動的に投稿されます。

📱

スマホ対応 UI

スマホのブラウザに最適化されたデザイン。外出中でもサクサク操作できます。

🖼️

画像の添付

投稿に画像を添付できます。JPEG・PNG・GIF・WebP に対応しています。

📋

テンプレート管理

よく使う文章をテンプレートとして保存。繰り返し使う投稿文の作成が素早くなります。

🖼️✨

サムネイル生成プロンプト

AI 向けのサムネイル生成プロンプトを自動作成。画像制作をもっとスムーズに。

🔒

ローカル完結・安全

データはすべて自分の PC に保存されます。クラウドへの情報流出はありません。

How it works

3ステップで始められます

インストールからスマホで使えるようになるまで、たった数分です。

1

パッケージをインストール

Python がインストールされていれば OK。1回だけ実行します。

pip install -r requirements.txt
2

Web アプリを起動

コマンドを実行するとローカルサーバーが立ち上がります。

python web_app.py
3

ブラウザで開く

PC は http://localhost:5000 へ。スマホは同じ Wi-Fi に接続して PC の IP アドレスにアクセスするだけです。
