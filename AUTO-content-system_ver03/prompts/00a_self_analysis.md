# Phase 0: 自己分析・棚卸し

## 概要

有料コンテンツの「種」を見つけるための自己分析フェーズ。
2つのモードを用意し、ユーザーの状況に合わせて選べるようにする。

**目標**: 自分の経験・スキル・変化を洗い出し、商品化できる「種」を3つ以上特定する

---

## 最初に表示: モード選択

```
============================================================
  Phase 0: 自己分析・棚卸し
============================================================

  モードを選んでください：

  1. ⚡ 簡易モード（おすすめ）
     → ジャンルを選ぶ → 2〜3問答えるだけ → 約1分で完了
     → テンプレートをベースにAIが自動補完します

  2. 📝 詳細モード
     → じっくりヒアリングしながら深掘り → 約10〜15分
     → あなただけのオリジナル素材を丁寧に発掘します
     → より精度の高いコンテンツを作りたい方におすすめ

============================================================
```

- **1を選択** → 簡易モード（PART A）へ進む
- **2を選択** → 詳細モード（PART B）へ進む

### STEP 1: ジャンル選択（選ぶだけ）

以下のメニューを表示し、ユーザーに選んでもらう。

```

  あなたのジャンルを選んでください
============================================================

  1. 恋愛・モテ・パートナーシップ
  2. 副業・お金・ビジネス
  3. スキルアップ・キャリア
  4. 健康・美容・ダイエット
  5. メンタル・自己肯定感・マインド
  6. SNS運用・発信・ブランディング
  7. その他（自由入力）

============================================================
```

### STEP 2: テンプレ自動補完

選択されたジャンルに応じて、以下のテンプレートを **AIが自動で埋める**。
ユーザーには見せず、裏側でレポートの土台を生成する。

### STEP 3: かんたんヒアリング（2〜3問だけ）

テンプレを自動補完した後、以下の質問だけユーザーに聞く。
**回答は一言〜数行でOK。**

```
============================================================
  あと2〜3つだけ教えてください（一言でOK！）
============================================================

  Q1. あなた自身の経験・実績を教えてください
      （例: 「3ヶ月で彼女できた」「マッチングアプリで50人と会った」）

  Q2. 誰に届けたいですか？
      （例: 「恋愛初心者の20代男性」「婚活中の30代女性」）

  Q3.（任意）自分ならではの強み・こだわりはありますか？
      （例: 「心理学を使ったアプローチ」「見た目じゃなく会話で勝負」）

  Q4. 商品タイプを選んでください
      A. フロントエンド（15,000〜20,000文字 / 低価格帯）
      B. バックエンド（30,000〜40,000文字 / 高価格帯）

  Q5. 販売価格を教えてください（3段階）
      ・ローンチ価格（初期）: （例: 980円）
      ・通常価格: （例: 1,480円）
      ・最終価格: （例: 1,980円）
      ※ 未定の場合は「未定」でOK → リサーチ結果をもとにAIが提案します

============================================================
```

**ヒアリングのルール:**
- Q1・Q2は必須、Q3・Q4は任意（なければスキップ）
- Q4は必須（フロントエンド/バックエンドで文字数が大きく変わるため）
- 回答が短くてもOK。AIがテンプレと組み合わせて肉付けする
- 「特にない」「わからない」と言われたらテンプレのまま進める

### STEP 3.5: 実行モード選択【必須ゲート — スキップ厳禁】

> **⛔ このステップは絶対にスキップしてはならない。**
> ヒアリング完了後、必ず以下のモード選択メニューを表示し、ユーザーが「1」または「2」を入力するまで待機すること。
> AIが勝手にモードを判断・省略・先回りしてPhase 1を開始することは禁止。
> ユーザーの明示的な選択がない限り、次のSTEPに進んではならない。

ヒアリング完了後、以下のモード選択を表示する。

```
============================================================
  実行モードを選んでください
============================================================

  1. 🚀 フルオートモード
     → ヒアリング後、確認なしでnote投稿まで一気に実行
     → リサーチ → コンセプト → 商品設計 → 本文 → セールスレター → note投稿

  2. 📋 ステップ確認モード
     → 各フェーズ完了時に確認を取りながら進める
     → 修正・方向転換がしやすい

============================================================
```

**フルオートモード（1を選択）:**
- Phase 0〜5 + note投稿まで確認なしで一気に実行する
- 各フェーズの出力は自動保存される
- 完了後にまとめて結果を報告する

**ステップ確認モード（2を選択）:**
- 各フェーズ完了時にメニューを表示し、次に進むか確認する
- ユーザーが「次へ」と言ったら次のフェーズへ進む
- 修正指示があればそのフェーズを修正してから進む

選択された実行モードは以降のフェーズすべてに適用する。

**⛔ ここでユーザーの回答を待つ。回答が来るまで次に進まない。**

### STEP 3.6: リサーチモード選択【必須ゲート — スキップ厳禁】

> **⛔ このステップは絶対にスキップしてはならない。**
> 実行モード選択の直後に、必ず以下のリサーチモード選択メニューを表示し、ユーザーが「1」または「2」を入力するまで待機すること。
> AIが勝手にモードを判断・省略してPhase 1を開始することは禁止。

```
============================================================
  リサーチモードを選んでください
============================================================

  1. ⚡ クイックリサーチ（おすすめ）
     → 3商品分析 / 軽量テンプレ / ネタ15個
     → 約5〜10分で完了

  2. 📝 詳細リサーチ
     → 5商品分析 / フル分析 / ネタ20個
     → 約15〜25分

============================================================
```

選択されたリサーチモードは Phase 1（競合リサーチ）で適用する。

**⛔ ここでユーザーの回答を待つ。回答が来るまでSTEP 4に進まない。**

### STEP 4: レポート生成

STEP 2のテンプレ + STEP 3のヒアリング回答を統合し、
自己分析レポートを自動生成する。

**統合ルール:**
1. ヒアリング回答がある項目 → テンプレを上書き・カスタマイズ
2. ヒアリング回答がない項目 → テンプレをそのまま使用
3. 掛け合わせ・商品の種 → ヒアリング回答を加味して再生成
4. セルフリサーチ（12項目）→ AIが全て自動補完

**1問ずつ会話形式で聞く**。一度に全部聞かない。

## 実行手順（詳細モード）

### STEP 1: ジャンル確認

まずジャンルを確認する（簡易モードと同じ選択肢）。

### STEP 2: ビフォーアフターの深掘り

以下を **1問ずつ順番に** 聞く。回答が出たら次の質問へ進む。

```
============================================================
  📝 詳細ヒアリング（1/5）ビフォーアフター
============================================================

  Q1. このテーマに取り組む前、どんな状態でしたか？
      （悩み・コンプレックス・困っていたこと）

  → 回答後 →

  Q2. 何がきっかけで変わり始めましたか？

  → 回答後 →

  Q3. 途中でぶつかった壁・挫折はありましたか？

  → 回答後 →

  Q4. 今はどんな状態ですか？どれくらいの期間で変わりましたか？

============================================================
```

### STEP 3: ストーリー素材の発掘

```
============================================================
  📝 詳細ヒアリング（2/5）ストーリー素材
============================================================

  Q5. このテーマで一番キツかった・恥ずかしかった経験は？

  → 回答後 →

  Q6. 「これはおかしい」「なんで誰も教えてくれないの？」
      と感じたことはありますか？

  → 回答後 →

  Q7. 「これだ！」と気づいた転機の瞬間は？

  → 回答後 →

  Q8. 一番嬉しかった成果・反応は？

============================================================
```

### STEP 4: 実績・数値の洗い出し

```
============================================================
  📝 詳細ヒアリング（3/5）実績・数値
============================================================

  Q9. 数字で示せる成果はありますか？
      （人数・期間・金額・フォロワー数など、なんでもOK）

  → 回答後 →

  Q10. 人に教えた・アドバイスした経験はありますか？
       その人はどうなりましたか？

============================================================
```

### STEP 5: スキル・独自性

```
============================================================
  📝 詳細ヒアリング（4/5）スキル・独自性
============================================================

  Q11. あなたの「やり方」で、
       他の人と違うところ・こだわりはありますか？

  → 回答後 →

  Q12. 「みんなはこうやるけど、自分はこうする」
       というアプローチはありますか？

============================================================
```

### STEP 6: 商品の種の確認

```
============================================================
  📝 詳細ヒアリング（5/5）方向性の確認
============================================================

  Q13. ここまでの回答をもとに、商品の種を3つ提案します。
       気になるもの・ピンとくるものはありますか？

  [AIが3つの種を提示]

  → ユーザーが選択 or 修正 →

  Q14. 商品タイプを選んでください
       A. フロントエンド（15,000〜20,000文字 / 低価格帯）
       B. バックエンド（30,000〜40,000文字 / 高価格帯）

  Q15. 販売価格を教えてください（3段階）
       ・ローンチ価格（初期）: （例: 980円）
       ・通常価格: （例: 1,480円）
       ・最終価格: （例: 1,980円）
       ※ 未定の場合は「未定」でOK → リサーチ結果をもとにAIが提案します

============================================================
```

### STEP 6.5: 実行モード選択【必須ゲート — スキップ厳禁】

> **⛔ このステップは絶対にスキップしてはならない。**
> ヒアリング完了後、必ず以下のモード選択メニューを表示し、ユーザーが「1」または「2」を入力するまで待機すること。
> AIが勝手にモードを判断・省略・先回りしてPhase 1を開始することは禁止。
> ユーザーの明示的な選択がない限り、次のSTEPに進んではならない。

ヒアリング完了後、簡易モードと同じ実行モード選択を表示する（PART A の STEP 3.5 と同じメニュー）。

**⛔ ここでユーザーの回答を待つ。回答が来るまで次に進まない。**

### STEP 6.6: リサーチモード選択【必須ゲート — スキップ厳禁】

実行モード選択の直後に、簡易モードと同じリサーチモード選択を表示する（PART A の STEP 3.6 と同じメニュー）。

**⛔ ここでユーザーの回答を待つ。回答が来るまでSTEP 7に進まない。**

### STEP 7: レポート生成

全ヒアリング回答をもとに、自己分析レポートを生成する。

**詳細モードの生成ルール:**
1. ユーザーの回答をそのまま活かす（言い換えすぎない）
2. 回答が薄い項目はAIが補完するが、必ず「※AI補完」と注記する
3. ストーリー素材はユーザーの生の言葉を最大限使う
4. セルフリサーチ（12項目）はヒアリング回答をもとにAIが自動生成

**出力フォーマットは簡易モードと同じ**（上記の出力フォーマットを使用）

## ヒアリングの進め方ルール（詳細モード共通）

1. **1問ずつ聞く** — 一度に複数の質問を投げない
2. **回答を受け止めてから次へ** — 「ありがとうございます！」等のリアクションを入れる
3. **深掘りは1回まで** — 回答が薄い場合、1回だけ掘り下げ質問してOK。それ以上はしつこい
4. **スキップOK** — 「わからない」「特にない」と言われたらスキップしてAI補完
5. **進捗を見せる** — 「（5問中3問目です）」のように進捗を表示する
