# 特典1: Claude Code × Threads投稿生成プロンプトテンプレート集（10パターン）

> コピペしてすぐ使える、ジャンル別の投稿生成プロンプト10種類。
> 気づき型・ストーリー型・リスト型・共感型・問いかけ型の全パターン対応。

---

## 使い方

各テンプレートの `【】` 内を自分の内容に書き換えてClaudeに貼り付けるだけです。

---

## パターン1：気づき型（Insight）

**概要**: 読者が「あ、そうか！」と思う発見・気づきを届ける投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: 気づき型
テーマ: 【例：料理のタイムパフォーマンスを上げる方法】
ターゲット: 【例：忙しい30代の料理初心者】
文字数: 150〜200文字
トーン: 親しみやすく、少し驚きがある
末尾: 「#時短料理 #気づき」などのハッシュタグを2〜3個

条件:
- 冒頭の1文で読者の注意を引く
- 「実は〜」「知らなかった〜」などの言葉を使う
- 具体的な数字やデータを1つ入れる
- 押しつけがましくしない
```

---

## パターン2：ストーリー型（Story）

**概要**: 自分の体験談・失敗談を通じて共感と信頼を得る投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: ストーリー型（体験談）
テーマ: 【例：料理が苦手だった私が毎日自炊できるようになった話】
ターゲット: 【例：料理に苦手意識がある人】
文字数: 200〜250文字
トーン: リアルで等身大、少し感情的
構成: 「以前の自分 → 転機 → 今の自分」の3段階

条件:
- 実際にあったような具体的なエピソードを入れる
- 感情の変化を表現する（困った→やってみた→変わった）
- 読者が「自分もできるかも」と思える終わり方にする
- ハッシュタグ2〜3個を末尾に追加
```

---

## パターン3：リスト型（List）

**概要**: 情報を箇条書きで整理して、保存・シェアされやすい投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: リスト型
テーマ: 【例：冷蔵庫の余り野菜を使い切る5つの方法】
ターゲット: 【例：食材を無駄にしがちな人】
文字数: 180〜220文字
トーン: 実用的でわかりやすい
構成: リードコピー（1文）+ 箇条書き（3〜5項目）+ 締めのひと言

条件:
- リスト項目は短く（各20文字以内）
- 番号または絵文字で視認性を上げる
- 「保存して使ってください」などの行動促進フレーズを入れる
- ハッシュタグ2〜3個を末尾に追加
```

---

## パターン4：共感型（Empathy）

**概要**: 読者の悩みや気持ちを代弁して「わかる！」を引き出す投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: 共感型
テーマ: 【例：仕事後に料理する気が起きない問題】
ターゲット: 【例：平日に自炊をがんばりたいが疲れてしまう人】
文字数: 150〜200文字
トーン: 優しく、寄り添う
構成: 共感の表明（1〜2文）+ 解決策のヒント（1〜2文）+ 背中を押す締め（1文）

条件:
- 「〜ですよね」「わかります」などの共感ワードを使う
- 読者を責めない、比べない
- 小さな一歩を提案する（ハードルを下げる）
- ハッシュタグ2〜3個を末尾に追加
```

---

## パターン5：問いかけ型（Question）

**概要**: 読者に問いを投げかけてコメントやリプライを促す投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: 問いかけ型
テーマ: 【例：みんなの「定番晩ごはん」を教えてほしい】
ターゲット: 【例：料理好き全般】
文字数: 100〜150文字
トーン: フレンドリー、会話的
構成: 問いかけ（1〜2文）+ 自分の答えを先に開示（1文）+ リプライ促進（1文）

条件:
- 答えやすい具体的な問いにする
- 「あなたはどうですか？」などの直接呼びかけを入れる
- 自分の意見を先に言うことで場を作る
- ハッシュタグ2〜3個を末尾に追加
```

---

## パターン6：ノウハウ型（How-to）

**概要**: 具体的なやり方・手順を教えて、専門性と信頼を示す投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: ノウハウ型（ステップ解説）
テーマ: 【例：失敗しない鶏むね肉のしっとり調理法】
ターゲット: 【例：料理中級者、健康志向の人】
文字数: 200〜250文字
トーン: 丁寧で信頼感がある
構成: タイトル的な1文 + ステップ（2〜4つ）+ プロのひとこと

条件:
- 「ポイントは〜」「コツは〜」など専門的な言葉を適度に使う
- 初心者でもわかる平易な表現にする
- 読者が「やってみたい」と思える具体性を入れる
- ハッシュタグ2〜3個を末尾に追加
```

---

## パターン7：比較型（Before / After）

**概要**: 変化・改善を見せることで説得力を高める投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: 比較型（Before/After）
テーマ: 【例：調味料を揃える前と後で料理の満足度がどう変わったか】
ターゲット: 【例：料理の質を上げたい人】
文字数: 180〜220文字
トーン: 体験ベース、説得力重視
構成: Before（1〜2文）+ After（1〜2文）+ 変化の理由（1文）+ 行動促進（1文）

条件:
- 数字や具体例で差を示す
- 読者が「自分も変えられる」と思える内容にする
- 大げさにしすぎない（リアリティを保つ）
- ハッシュタグ2〜3個を末尾に追加
```

---

## パターン8：トレンド活用型（Trend）

**概要**: 今話題のテーマに乗っかって拡散力を高める投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: トレンド活用型
テーマ: 【例：「節約飯」ブームに乗じて低コスト高タンパクレシピを紹介】
ターゲット: 【例：節約・健康意識が高い20〜30代】
文字数: 150〜200文字
トーン: タイムリー感があり、軽快
構成: トレンドへの言及（1文）+ 自分の切り口（1〜2文）+ 具体的な提案（1〜2文）

条件:
- 流行ワードを自然に取り込む
- 「今こそ〜」「最近話題の〜」などのフレーズを使う
- 差別化のある独自の視点を1つ入れる
- ハッシュタグ2〜3個（トレンドタグ含む）を末尾に追加
```

---

## パターン9：引用・名言型（Quote）

**概要**: 印象的な言葉を起点に自分の意見を展開する投稿

```
以下の内容でThreads投稿を生成してください。

スタイル: 引用・名言型
テーマ: 【例：「料理は愛情」という言葉について自分の考えを述べる】
ターゲット: 【例：料理と暮らしに興味がある人全般】
文字数: 150〜200文字
トーン: 思慮深く、少し詩的
構成: 引用または印象的な一文（1文）+ 自分の解釈・意見（2〜3文）+ 読者への問いかけ（1文）

条件:
- 引用は実在または架空どちらでもよい（架空の場合は「ある人の言葉」などとする）
- 自分の言葉で言い換えた解釈を加える
- 抽象的すぎず、生活に根ざした内容にする
- ハッシュタグ2〜3個を末尾に追加
```

---

## パターン10：シリーズ・連投型（Series）

**概要**: 「Day1」「Week1」など続きがある投稿で習慣的なフォローを促す

```
以下の内容でThreads投稿を生成してください。

スタイル: シリーズ連投型（初回 or 継続回）
シリーズ名: 【例：「30日自炊チャレンジ」】
今回の回数: 【例：Day 7】
今日のトピック: 【例：冷凍食材を上手に使う方法を実践した】
ターゲット: 【例：同じチャレンジに挑戦中の人・興味がある人】
文字数: 150〜200文字
トーン: 日記的、親近感
構成: 何日目かの宣言（1文）+ 今日の出来事・気づき（2〜3文）+ 次回予告（1文）

条件:
- 毎日読みたくなる「続き感」を出す
- 失敗や迷いも正直に書く（完璧を演じない）
- 読者も参加できる問いかけを入れる
- シリーズタグ＋関連ハッシュタグ2〜3個を末尾に追加
```

## ワンポイントアドバイス

| パターン | 向いているゴール |
|---------|----------------|
| 気づき型 | インプレッション・保存数アップ |
| ストーリー型 | フォロワーとの信頼関係構築 |
| リスト型 | 保存・シェア数アップ |
| 共感型 | コメント・リプライ促進 |
| 問いかけ型 | エンゲージメント率アップ |
| ノウハウ型 | 専門性・権威性の向上 |
| 比較型 | 説得力・信頼度アップ |
| トレンド型 | 新規リーチ・拡散 |
| 引用型 | フォロワーの思考促進 |
| シリーズ型 | 継続フォロー・定着率アップ |
